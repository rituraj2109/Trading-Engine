# Trading-Engine

Forex AI Decision Engine

This repository contains a Forex market signal generator that:
- Fetches market data every 15 minutes
- Calculates technical indicators and detects chart patterns
- Updates news sentiment
- Generates BUY/SELL/WAIT signals and stores them in a local SQLite DB

See `DEPLOYMENT_READY.txt` and `VERIFICATION_SUMMARY.txt` for deployment and verification details.
